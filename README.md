# LdySSH 🌌

**一款次世代、极富科幻美学的全能型智能 SSH 客户端**

LdySSH 是一款融合了 **3D 集群网络拓扑**、**本地离线大模型 AI 辅助**、**极速多会话终端** 以及 **可视化 SFTP 文件管理** 的新一代智能化桌面客户端。我们采用精美的毛玻璃特效、星空微光设计和极简的交互界面，旨在为开发者提供最纯粹、最科幻、最高效的终极终端操作体验。

---

## ✨ 核心特色功能 (Key Features)

### 🌐 1. 三维星尘集群拓扑看板 (3D Cluster Network Topology)
*   **星系渲染中枢**：集成 Three.js 高性能 WebGL 渲染管线，将您连接过的所有远程主机映射为 3D 星空中的恒星与行星轨道。
*   **实时延迟监测**：支持实时 Ping 探测，通过恒星光晕和连线颜色的明暗呼吸，直观呈现集群中各主机的健康度与通信延迟。
*   **多维交互体验**：完美支持鼠标左键旋转视角、右键平移星系、滚轮缩放宇宙，实现人机交互的极致科幻感。

### 🤖 2. 内置本地离线 AI 助手 (Hermes Local AI Chat)
*   **无网纯离线推理**：内置轻量级、高性价比的本地大模型推理引擎（`llama-server.exe` 驱动），数据完全本地留存，绝对保障隐私安全。
*   **极简流畅体验**：前后台所有子进程均通过 Windows 作业对象（Job Object）进行生命周期强绑定，并且通过全局 `subprocess` 隐藏技术，**完全屏蔽任何 CMD 终端黑色窗口的闪烁**。
*   **智能下载与热加载**：在大模型未就绪时，支持以 2 秒频率动态呈现极速流式下载百分比及网络带宽；下载完成后，系统会自动关闭 Mock 进度服务，极速热拉起真实的推理服务器，告别聊天超时。

### 🎨 3. 精致紧凑的主机工作台 (Sleek empty-state Workbench)
*   **极简扁平设计**：摒弃了笨重、占空间的虚线框卡片，在主机数量为 0 时，自动淡入小巧且极富质感的扁平呼吸发光 `➕ 新建连接` 按钮，界面布局更紧凑。
*   **智能穿透技术**：大厅背景融入了 3D 宇宙尘埃粒子动画，结合精妙的鼠标事件穿透技术，点击按钮与划动背景星空互不干扰。

### 🖥️ 4. 极速多会话终端与系统监控 (Modern Terminal & System Monitor)
*   **多会话标签页**：基于现代 Web 终端技术的极速会话管理器，支持多窗口、多主机自由切换，自动重连。
*   **一体化系统监控**：实时以可视化图表动态呈现远程主机的 CPU 占用、内存负荷、磁盘容量与进程管理器，掌控全局。
*   **可视化 SFTP**：支持拖拽上传、双击下载、目录树浏览，让复杂的网络文件传输像本地操作一样简单。

## 🚀 快速开始 (Getting Started)

### 1. 使用绿色发布版（推荐，一键运行）
直接进入项目发布目录，双击运行即可：
```bash
LdySSH_Release/LdySSH.exe
```
> **提示**：运行后系统会自动初始化 Web 容器及离线 AI 服务，客户端关闭时，后台所有子进程会被 Windows Job 自动强退释放，无任何后台残留。

### 2. 源码开发与运行
如果您想进行二次开发，请确保您的系统安装了 Python 3.8+ 及 C++ 编译环境：

```bash
# 1. 安装核心依赖
pip install -r requirements.txt

# 2. 运行 Python 混合容器后端
python src/main.py
```

## 🛠️ 项目架构说明 (Architecture)

*   **`src/`**：Python 核心后端逻辑，负责 SSH 协议握手、SFTP 通信、系统监控数据采集、API 路由分配等。
*   **`src/ui/`**：基于 Vanilla CSS + HTML5 打造的极致科技感前端大厅，集成了 Three.js 拓扑及 xterm.js 终端。
*   **`prismssh-cpp/`**：C++ 桌面壳程序，利用 Windows 内核 API 与 Job Object 实现了对后台 Python 及 LLM 进程的静默管理与极速清退。
*   **`upstream-hermes-webui/`**：集成在客户端内的极简 AI 对话前端，专为开发者量身定制。
